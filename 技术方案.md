# 智能计算系统大作业：技术方案说明

## 1. 架构总览

双层检索 + LLM 增强生成（已落地）。

```
问题
  │
  ▼
Qwen Embedding → 编码为句向量
  │
  ├─ Stage1: QA 库精确匹配 ─── 相似度 > 0.8 ──→ 直接输出标准答案
  │
  └─ Stage2: 知识库检索 + LLM 生成
       │
       ├─ 知识库 Top-5 (相似度 > 0.5) → 约束 Prompt + Qwen2.5-3B 生成
       └─ 无命中 → 退化 LLM 自主思考模式
```

## 2. 模型

| 模型 | 用途 | 路径 |
|------|------|------|
| Qwen Embedding | 文本 → 句向量 | `model/qwen_embedding_model/` |
| Qwen2.5-3B | 知识增强生成 | `model/qwen2.5-3B/` |

- 嵌入模型：全局单例，首次调用后缓存
- LLM：延迟加载（仅 Stage2 触发），`do_sample=True`, temperature=0.7, top_p=0.9, max_new_tokens=2048

## 3. 知识库

### 3.1 QA 库 (`rag/embedding_qa.json`)
- 306 条，公开思考题 + 标准答案
- Stage1 原题匹配（拿时间分）

```json
{
  "npyID": "0",
  "Question": "...",
  "Answer": "..."
}
```

### 3.2 知识库 (`rag/embedding_knowledge.json`)
- 1121 条，课程文档清洗切片
- Stage2 知识片段检索

```json
{
  "npyID": "0",
  "source": "第2章-神经网络设计实验",
  "content": "具体的课程知识点..."
}
```

### 3.3 向量文件
- `.npy` 由 `tools/make_npy.py` 离线生成
- `.gitignore` 排除 `rag/*.npy`
- float32 格式，L2 归一化

## 4. 检索流程

1. **编码**：Qwen Embedding → last token 隐状态 → L2 归一化
2. **余弦相似度**：`candidates @ query` 矩阵乘法（`np.linalg.norm` 归一化后内积）
3. **Stage1**：QA 库最高分 > 0.8 → 直接返回 Answer
4. **Stage2**：
   - 知识库 Top-5（相似度 > 0.5）→ 约束 Prompt → LLM 生成
   - 无命中 → 退化 Prompt → LLM 自主思考
   - 知识库不可用 → 报错

## 5. Prompt 设计

约束模式（有知识）：严格基于给定知识，不编造，不提评分/排名，1500 字以内
退化模式（无知识）：基于 LLM 自身知识，约束同上

## 6. 阈值

| 参数 | 值 | 说明 |
|------|-----|------|
| QA_THRESHOLD | 0.8 | QA 库命中 |
| KNOWLEDGE_THRESHOLD | 0.5 | 知识库检索 |
| KNOWLEDGE_TOP_K | 5 | 最多取几条 |

## 7. 工程优化

### 静默模式（默认）
- `TQDM_DISABLE=1`, `HF_HUB_DISABLE_PROGRESS_BARS=1`
- `HF_HUB_OFFLINE=1`, `TRANSFORMERS_OFFLINE=1`
- stderr → /dev/null 重定向（屏蔽 torch 版本警告等噪音）
- transformers logging set_verbosity_error

### 调试模式
`python main.py "问题" --debug` → 相似度排名 + 命中详情输出到 stderr

## 8. 打包规范

### ZIP 结构
```text
model.zip
├── main.py
├── model/
│   ├── qwen_embedding_model/
│   └── qwen2.5-3B/

```

### 排除
- `data/`、`tools/`、`plans/` 不入包
- 模型权重 `model/`、向量 `rag/*.npy`、`*.zip` 皆入 `.gitignore`

### 运行接口
```bash
python main.py "问题文本"
```
- `sys.argv[1]` 读问题，stdout 输出回答
- 无交互输入，无文件写入
