## 7.2.1 实验目的

## 7.2.2 背景介绍

### 7.2.2.1 目标检测

目标检测对图像中的特定类别目标（例如人、动物、车等）进行定位并分类。目标检测是实例分割、图像描述、目标跟踪等计算机视觉任务的基础，目前已广泛应用于自动驾驶、机器人视觉、视频分析等领域。

基于深度学习的目标检测算法主要包括两阶段算法和一阶段算法：

- **两阶段算法**：基于候选区域提取方法，首先产生候选框将所有可能的潜在目标框出来，然后用基于卷积神经网络的图像分类算法对候选框进行分类和进一步的回归，获得最终的边界框（Bounding Box）。代表性算法包括R-CNN系列。
- **一阶段算法**：仅用卷积神经网络直接输出目标位置坐标和分类。代表性算法包括YOLO、SSD等。

经过多个版本的改进，YOLO在模型架构、训练策略、预测技巧等方面进行了不断的升级，提高了模型的检测性能和速度。YOLOv5目标检测实验选用经典的一阶段检测算法YOLOv5网络做目标检测。

### 7.2.2.2 YOLOv5网络

与其他YOLO系列的网络相比，YOLOv5的目标检测思想并没有发生太大的变化，主要是网络结构的变更，以及更灵活的参数控制方式。其基本检测流程如下：利用神经网络提取多尺度特征融合，并预测边界框的位置和类别，最后利用NMS筛选出合适的输出。

YOLOv5的官方代码中，给出了5个版本的目标检测网络，分别是YOLOv5n、YOLOv5s、YOLOv5m、YOLOv5l、YOLOv5x五个模型。和之前的YOLOv3、YOLOv4中的cfg文件不同，YOLOv5代码中的网络都是以yaml的形式来呈现的。YOLOv5n网络是YOLOv5系列中深度最小、特征图的宽度最小的网络。后面的4种都是在此基础上不断加深、不断加宽，分别对应yaml文件中的depth_multiple和width_multiple参数。

以YOLOv5m为例，其主要由输入端、网络结构、输出端三部分组成：

**1) 输入端**

YOLOv5m中采用了一些高级的数据增强策略，包括Mosaic数据增强、自适应锚框计算和自适应图片缩放策略，这些策略有助于提高模型的性能和鲁棒性。

数据增强策略包括：
- **Mixup数据增强**：从每个Batch中随机选择两张图片，并以一定比例混合生成新的图像，训练过程全部采用混合的新图像训练，原始图像不再参与训练。
- **Cutout数据增强**：在训练时随机地屏蔽输入图像中的方形区域。
- **CopyPaste数据增强**：将部分目标随机地粘贴到图片中，前提是数据要有segments数据才行，即每个目标的实例分割信息。
- **Random Affine仿射变换**：包含随机旋转、平移、缩放、错切（将所有点沿某一指定方向成比例地平移）、透视操作等。
- **HSV随机增强图像**：通过HSV增强获得色调、饱和度、亮度的多样性，使模型在训练过程中看到的数据更加多样。
- **随机水平翻转**：将图片进行随机翻转。

自适应锚框计算根据数据集中的目标尺寸和分布自动计算锚框的尺寸和长宽比，这样可以更好地适应不同数据集和目标类型，避免了手动选择锚框大小和比例的繁琐过程，有助于提高检测框的准确性。

自适应图片缩放策略根据训练数据中的目标尺寸来自动调整输入图像的大小。这有助于确保模型能够有效地处理不同大小的目标，而无需将所有图像都调整为相同的尺寸，提高了模型的适应性和泛化能力。

**2) 网络结构**

网络结构主要由Backbone、Head组成。

**Backbone（骨干网络）**是模型的主要特征提取部分，由Conv（CBS）、C3以及SPPF组成：

- **Conv模块**：由卷积层-批归一化层-SiLU层组成。SiLU函数可以看作是Sigmoid函数的一种扩展形式，它保留了Sigmoid的非线性特性，同时具有更平滑的梯度。SiLU的计算公式为：$SiLU(x) = x \cdot \sigma(x)$，其中 $\sigma$ 为Sigmoid函数。
- **C3模块**：YOLOv5中的关键组成部分，有助于提高检测性能并降低计算成本。它主要由Conv、Bottleneck1以及Concat操作组成。该模块包含两支，一支使用了Conv层以及多个Bottleneck堆叠（Bottleneck1采用残差连接方式），另一支仅经过一个基本Conv模块，最后将两支进行Concat操作。
- **SPPF模块**：一种特征处理组件，用于捕获多尺度的目标特征。它包括一个SPP（Spatial Pyramid Pooling）池化层，将输入特征图分成不同尺度的网格，然后对每个网格进行池化。与传统SPP不同，SPPF还引入了融合特征的操作，将不同尺度的池化结果与原始特征图相融合，以增强多尺度特征的表示能力。它经过一次卷积，然后三次池化（MaxPool2d with kernel_size=5），分别将每次池化结果拼接起来后再进行一次卷积。

**Head**采用了一种特征金字塔网络（FPN+PAN结构），用于整合不同层次的特征以提高目标检测性能。它由Conv（CBS）、C3模块以及Upsample、Concat操作构成。其中，Conv和C3模块与Backbone中的结构相似，只不过C3中的Bottleneck1替换为Bottleneck2（不采用残差连接）。通过将来自不同层级的特征图进行融合和上采样，以产生多尺度的特征金字塔。这有助于模型捕获不同目标尺度的特征，提高检测精度。Head中还包括横向连接，将底层和高层特征图连接在一起，从而增强了特征融合的效果。

**3) 输出端**

YOLOv5的输出端是目标检测流程的核心，负责从模型输出中提取目标检测结果。这一部分主要包括三个关键组件：Decode（解码）、NMS（非极大值抑制）、TopK和损失函数。

- **Decode（解码）**：将模型输出的预测信息转化为真实世界中的目标框的过程。模型输出的通常包括目标框的坐标、类别置信度分数以及类别标签。解码操作将这些信息与模型训练时使用的锚框（Anchor Boxes）结合起来，从而得到目标框的具体位置和类别。解码过程通常包括对坐标进行逆标准化、应用Sigmoid激活函数、计算框的坐标和置信度得分等步骤。
- **NMS（非极大值抑制）**：一种用于去除冗余检测框的技术，它确保每个目标只被保留一个检测框，以消除重叠检测。NMS操作首先根据类别置信度对所有检测框进行排序，然后从置信度最高的框开始，逐一与后续框进行比较。如果两个框之间的IoU（交并比）大于某个阈值（通常是0.5），则较低置信度的框将被抑制，只保留置信度更高的框。不同于YOLOv4的Diou-NMS，YOLOv5中采用加权NMS的方式。
- **TopK操作**：为了限制输出的检测结果数量，通常会选择置信度分数最高的前K个检测框，K是一个用户定义的参数。这有助于减少处理时间和资源需求，特别是在需要高效的实时应用中。

YOLOv5使用的损失函数是一种组合损失函数，包括三个部分：

1. **置信度损失（Confidence Loss）**：衡量检测框与真实目标框之间的匹配程度。包含目标存在损失和目标不存在损失两个部分，使用二元交叉熵损失函数进行计算。
2. **框位置损失（Box Localization Loss）**：衡量预测框的位置和尺寸与真实目标框的位置和尺寸之间的差异。YOLOv5使用的是CIoU损失函数来计算框位置损失。
3. **类别损失（Class Loss）**：衡量预测框的类别标签与真实目标框的类别标签之间的差异。使用交叉熵损失函数来计算类别损失。

这三个部分的损失函数通过加权求和得到最终的总损失。权重系数可以根据实际需求进行调整，以平衡不同部分对总损失的贡献。

## 7.2.5 实验步骤

### 7.2.5.1 输入端：自适应图片缩放策略

在项目实际使用时，很多图片的长宽比不同，因此缩放填充后，两端的黑边大小都不同，而如果填充的比较多，则存在信息冗余，影响推理速度。因此在YOLOv5的代码letterbox函数中进行了修改，对原始图像自适应的添加最少的黑边。

该算法主要用于将输入图像调整为指定大小并进行填充，以满足深度学习模型对输入图像大小和步幅的要求。根据函数的参数设置，可以选择是否进行缩放、填充、拉伸等操作。

以1280×800的图片得到640×640大小的图片为例，算法分为三个步骤：

1. **计算缩放比例**：长的缩放比例为640/1280=0.5，宽的缩放比例为640/800=0.8，选择较小的缩放系数0.5。
2. **计算缩放后的尺寸**：原始图片的长宽都乘以最小的缩放系数0.5，宽变成了400，高变成了640。
3. **计算黑边填充数值**：将640-400=240，得到原本需要填充的高度。如果auto参数为True，则将填充调整为使宽度和高度都成为步幅stride的整数倍（使用np.mod函数计算）；如果scaleFill参数为True，则将图像拉伸以填充新形状，将填充量设置为0，同时调整比例以匹配新形状。将填充均匀分成两侧，以便在图像的四个边上添加填充。

如果输入图像的形状与新的未填充形状不匹配，使用线性插值方法将图像重新调整大小，使其匹配新形状。计算上、下、左、右填充的像素数，四舍五入以确保填充像素数为整数。最后使用OpenCV的`cv2.copyMakeBorder`函数在图像的四个边上添加填充。

核心代码逻辑如下：

```python
# augmentations.py — 自适应图片缩放策略
def letterbox(im, new_shape=(640, 640), color=(114, 114, 114), auto=True,
              scaleFill=False, scaleup=True, stride=32):
    # 计算缩放倍数
    shape = im.shape[:2]  # 当前图片尺寸 [height, width]
    r = min(new_shape[0] / shape[0], new_shape[1] / shape[1])
    if not scaleup:  # 只缩小不放大
        r = min(r, 1.0)

    # 计算实际缩放的结果大小
    new_unpad = (int(round(shape[1] * r)), int(round(shape[0] * r)))
    # 计算宽度和高度方向上的填充量
    dw, dh = new_shape[1] - new_unpad[0], new_shape[0] - new_unpad[1]

    # 如果auto参数为True，将填充调整为使宽度和高度都成为步幅的倍数
    if auto:
        dw, dh = np.mod(dw, stride), np.mod(dh, stride)
    elif scaleFill:  # 拉伸填充
        dw, dh = 0.0, 0.0
        new_unpad = (new_shape[1], new_shape[0])
        ratio = new_shape[1] / shape[1], new_shape[0] / shape[0]

    dw /= 2  # 将填充均分到两侧
    dh /= 2

    # 使用cv2.resize进行缩放
    if shape[::-1] != new_unpad:
        im = cv2.resize(im, new_unpad, interpolation=cv2.INTER_LINEAR)
    top, bottom = int(round(dh - 0.1)), int(round(dh + 0.1))
    left, right = int(round(dw - 0.1)), int(round(dw + 0.1))

    # 使用cv2.copyMakeBorder在图像四边添加填充
    im = cv2.copyMakeBorder(im, top, bottom, left, right,
                            cv2.BORDER_CONSTANT, value=color)
    return im, ratio, (dw, dh)
```

### 7.2.5.2 网络结构

**Conv模块：** Conv模块包含一个卷积层（Conv2d）、一个批归一化层（BatchNorm2d）和一个SiLU激活函数。在正向传播方法forward中，输入x依次通过卷积、批归一化和激活函数处理，然后返回处理后的结果。同时，还提供了一个名为forward_fuse的方法，与forward类似，但在卷积之后直接应用激活函数（跳过批归一化），适用于某些推理优化需求。

```python
# common.py — Conv模块
class Conv(nn.Module):
    def __init__(self, c1, c2, k=1, s=1, p=None, g=1, act=True):
        super().__init__()
        self.conv = nn.Conv2d(c1, c2, k, s, autopad(k, p), groups=g, bias=False)
        self.bn = nn.BatchNorm2d(c2)
        self.act = nn.SiLU() if act is True else (act if isinstance(act, nn.Module) else nn.Identity())

    def forward(self, x):
        return self.act(self.bn(self.conv(x)))

    def forward_fuse(self, x):
        return self.act(self.conv(x))
```

**C3模块：** C3模块由Conv模块、Bottleneck1模块构成，是YOLOv5中的关键特征提取组件。其结构包含两条分支：第一条分支经过一个Conv模块后连接多个Bottleneck1模块（数量由n参数决定）；第二条分支仅经过一个Conv模块。两条分支的输出在通道维度上进行Concat拼接，然后再经过一个Conv模块得到最终输出。

其中，Bottleneck1采用残差连接的方式：输入经过"Conv(1×1)→Conv(3×3)"后与原始输入相加。这种残差结构有助于梯度传播和深层网络训练。

**SPPF模块：** SPPF层是用于YOLOv5等目标检测模型的一种快速空间金字塔池化层。它将输入特征图分别池化成不同尺度的子区域，然后将这些子区域特征拼接在一起，以提供多尺度的信息。具体实现为：输入首先通过一个Conv模块（kernel_size=1），然后依次进行三次MaxPool2d（kernel_size=5, stride=1, padding=2，输出尺寸不变），将原始特征和三次池化后的特征在通道维度上拼接，最后再通过一个Conv模块输出。这有助于模型更好地检测不同尺寸的物体。

### 7.2.5.3 输出端：NMS非极大值抑制

非极大值抑制（NMS）是目标检测算法中的重要环节之一。通过特征提取网络得到的边界框，在同一物体位置可能会有多个边界框，而这些边界框可能会有重叠。因此需要采用NMS来去除冗余的边界框，从而提高检测精度。

在一般情况下对每个类别做NMS处理的过程如下：

1. 将所有边界框按照置信度score进行排序并作为候选框列表；
2. 根据排序结果依次处理置信度最高的边界框，将其作为正确预测框输出，同时将其从候选框列表中移除；
3. 计算上一步中的边界框与其他边界框的交并比IoU，从候选框列表中移除重叠度高于等于阈值$t_{iou}$的框，即移除$IoU \geq t_{iou}$的边界框；
4. 重复上述两个步骤，直到没有置信度大于$t_{score}$的候选框为止。

其中，交并比（Intersection over Union，IoU）的计算公式为：

$$IoU = \frac{|A \cap B|}{|A \cup B|}$$

其中$A$和$B$为两个边界框，分子为两框相交的面积，分母为两框相并的面积。

YOLOv5的NMS实现核心逻辑（non_max_suppression函数）：

1. **设置参数**：min_wh=2, max_wh=4096（像素级最小/最大框宽高），max_nms=30000（最大NMS处理框数），time_limit=10.0（超时限制）。
2. **遍历每张图像**：对每张图像的预测结果进行处理。
3. **置信度筛选**：使用置信度阈值conf_thres筛选候选框，筛选出具有足够置信度的预测结果。
4. **框格式转换**：将框格式从（中心x、中心y、宽度width、高度height）转换为（x1, y1, x2, y2）格式。
5. **类别处理**：如果multi_label为True，获取每个目标框的所有超过阈值的类别；否则获取每个目标框的最大置信度及对应类别索引。
6. **按置信度排序**：根据目标框的置信度降序排列，选择前max_nms个具有最高置信度的目标框。
7. **批量NMS**：调用`torchvision.ops.nms`进行非极大值抑制。
8. **加权NMS合并**（可选）：调用box_iou计算目标框之间的IoU矩阵，通过iou_thres阈值进行筛选，将大于阈值的IoU设为True，使用加权平均合并重叠框。
9. **限制检测数量**：最多保留max_det个检测结果。

## 7.2.6 推荐实践内容

- **基础实践**：补全输入端的自适应图片缩放策略，得到正确大小的输入图片。
- **进阶实践**：在基础实践的基础上，补全网络结构模块，正确实现Conv、C3、SPPF模块的代码。
- **高级实践**：在进阶实践的基础上，补全输出端的NMS非极大值抑制模块，去除冗余的边界框，能够完成直接训练、预训练、恢复训练三种训练方式，并进行精度验证。
