## 4.4.2 背景介绍

### 4.4.2.1 hsigmoid介绍

激活函数是深度神经网络中每一个神经元线性加权计算结果的非线性处理，赋予神经网络非线性映射能力。常见的激活函数有Sigmoid、Tanh、ReLU、Leaky ReLU等。

其中，Sigmoid函数具有以下优点：

1. 输出值在0到1之间，方便观察模型的输出是否符合预期；
2. Sigmoid函数的导数在0到0.25之间，可以控制反向传播过程中权值的更新幅度；
3. Sigmoid函数的输出接近0或1时，输出的导数接近0，可以避免梯度消失的问题。

本实验自定义hsigmoid算子实现参照Sigmoid激活函数，其具体计算公式如下：

$$hsigmoid(x) = \frac{1}{1 + e^{-x}}$$

该算子的计算流程为：对输入张量中的每个元素，首先计算指数函数 $e^{-x}$，然后计算分母 $1 + e^{-x}$，最后计算倒数得到输出。输入和输出张量的形状保持一致。

### 4.4.2.2 添加PyTorch算子流程

深度学习算法由一个个计算单元组成，我们将这些计算单元称为算子（Operator，简称Op）。在PyTorch中添加新算子主要分为三种方式：基于原生PyTorch的方式、基于C++ Extension的方式以及基于OP Register方式添加算子。

- **基于原生PyTorch的方式**：这是PyTorch原生算子的组织方式。这种方式可以比较便捷地增加或者修改算子，但是需要在PyTorch源码中修改，与PyTorch耦合度高，每次新增或修改都要重新编译PyTorch。
- **C++ Extension以及Op Register的方式**：与PyTorch相互解耦，增加算子不需要修改PyTorch源码。

本实验对C++ Extension的方式实现算子添加进行介绍，该方式一般包括以下三个步骤：

1. **C++实现**：需要用C++语言实现算子的具体功能。算子的实现称为kernel。针对不同的输入、输出类型或硬件架构（如CPU、GPU或DLP等），可以有多个kernel实现。
2. **算子编译**：将C++算子实现编译成库文件。具体有三种方式：通过CMake编译、通过torch的JIT编译以及通过setuptools编译。本实验采用setuptools的编译方式。
3. **算子测试**：通过编写测试程序来测试采用C++实现的算子是否添加成功。

### 4.4.5.1 C++实现

自定义hsigmoid算子的C++实现代码（hsigmoid.cpp）包含头文件引用、hsigmoid算子的具体实现和算子绑定为PyTorch的模块三个部分：

1. **头文件引用**：需要包含`torch/extension.h`头文件，这是编写PyTorch的C++扩展时必须包含的一个文件。它基本上囊括了实现中所需要的所有依赖，包含了ATen库（PyTorch底层张量运算库）、pybind11（实现C++代码到Python的绑定）和二者之间的交互。

2. **hsigmoid算子的具体实现**：函数签名为`torch::Tensor hsigmoid_cpu(const torch::Tensor& dets)`。实现流程为：
   - 先将传入的Tensor数据通过`.contiguous()`确保内存连续性，然后转换为`std::vector<float>`类型的vector；
   - 遍历vector中的每个元素，根据hsigmoid的计算公式 $y = 1/(1+e^{-x})$ 计算结果；
   - 将输出数据通过`torch::from_blob`重新转换为Tensor数据类型，使用`torch::TensorOptions().dtype(torch::kFloat32)`指定数据类型为float32；
   - 最后将输出Tensor的形状reshape为`[1, 3, 512, 512]`。

3. **算子绑定为PyTorch的模块**：需要增加pybind11的绑定说明。这是由于PyTorch的C++ Extension是通过pybind11绑定到Python的。使用`PYBIND11_MODULE`宏定义模块，在`m.def`中传入参数，分别是绑定到Python的函数名称、需绑定的C++函数引用，以及一个简短的函数说明字符串，用来添加到Python函数中的`__doc__`成员名称中。

```cpp
// hsigmoid.cpp — PyTorch扩展头文件的引用
#include <torch/extension.h>
using namespace std;

// hsigmoid_cpu函数的具体实现
torch::Tensor hsigmoid_cpu(const torch::Tensor& dets) {
    // 将输入的tensor转化为浮点类型的vector
    auto input_data = dets.contiguous().data_ptr<float>();
    int input_size = dets.numel();
    
    // 创建一个浮点类型的output_data，大小与输入相同的vector
    std::vector<float> output_data(input_size);
    
    // 对于输入向量的每个元素计算hsigmoid: y = 1/(1+e^{-x})
    for (int i = 0; i < input_size; i++) {
        output_data[i] = 1.0f / (1.0f + std::exp(-input_data[i]));
    }
    
    // 创建TensorOptions，指定dtype为float32
    auto opts = torch::TensorOptions().dtype(torch::kFloat32);
    // 从output vector创建tensor并clone
    auto foo = torch::from_blob(output_data.data(), {int64_t(output_data.size())}, opts).clone();
    // 将得到的tensor reshape为所需的大小[1, 3, 512, 512]
    auto output = foo.reshape({1, 3, 512, 512});
    
    return output;
}

// 算子绑定为PyTorch的模块
PYBIND11_MODULE(TORCH_EXTENSION_NAME, m) {
    m.def("hsigmoid_cpu", &hsigmoid_cpu, "Custom hsigmoid operator on CPU");
}
```

### 4.4.5.2 算子编译

本实验利用Python中提供的setuptools模块完成编译流程，将写有算子的C++文件编译成为一个动态链接库（在Linux平台是一个.so后缀文件），从而让Python调用其中实现的函数功能。

setup.py文件的编写方式为：使用`setuptools.setup()`函数，通过`CppExtension`来包含待编译的文件及编译函数，并对编译命令进行设置。`CppExtension`的参数包括模块名称（编译后的库名）和源文件列表（包含hsigmoid.cpp的路径）。编译时使用`python setup.py build_ext --inplace`命令，在当前目录生成.so动态链接文件。

```python
# setup.py — 算子编译代码
from setuptools import setup
from torch.utils.cpp_extension import CppExtension, BuildExtension

setup(
    name='hsigmoid_cpu',
    ext_modules=[
        CppExtension(
            name='hsigmoid_cpu',       # 编译链接库的名称
            sources=['hsigmoid.cpp'],   # 待编译的C++源文件
        )
    ],
    cmdclass={
        'build_ext': BuildExtension    # 使用PyTorch的BuildExtension编译命令
    }
)
```

编译命令：

```bash
python setup.py build_ext --inplace
```

执行后会在当前目录生成`hsigmoid_cpu.so`动态链接文件。

### 4.4.5.3 算子测试

为了测试Python是否可以正确导入动态链接库中的hsigmoid算子，通过测试程序test_hsigmoid.py对算子进行测试。

测试流程如下：
1. 导入必要的库和模块（numpy、torch），以及自定义的动态链接库`hsigmoid_cpu`；
2. 定义一个在CPU上执行的hsigmoid函数作为包装函数；
3. 在测试函数中生成一个形状为(3, 512, 512)的随机张量；
4. 调用hsigmoid_cpu函数对其进行处理得到输出结果；
5. 打印输入张量和输出结果的形状，验证形状是否一致。

```python
# test_hsigmoid.py — 算子测试代码
import numpy as np
import torch
import torchvision
import numpy as np
# 导入自定义连接库
import hsigmoid_cpu

def hsigmoid_cpu_wrapper(rand):
    rand = rand.contiguous()
    # 调用hsigmoid函数对rand进行处理得到输出结果
    output = hsigmoid_cpu.hsigmoid_cpu(rand)
    return output

# 测试：生成随机张量并验证
if __name__ == '__main__':
    rand = torch.rand(3, 512, 512)
    result = hsigmoid_cpu_wrapper(rand)
    print(f"输入张量形状: {rand.shape}")
    print(f"输出结果形状: {result.shape}")
```
